Step 1

League Setup

Go to Admin Panel → League Setup tab. Configure everything before importing players.

Global Settings

Number of TeamsSquad Size Starting BudgetMin Bid

Teams

Add one row per team. Set a team name and password — share these with each team owner so they can log in.

Player Pools

Players are grouped into pools by skill tier (A1, A2 … D). Each pool has a base price and a count.

⚠️ The total player count across all pools must equal Teams × Squad Size.

👑 Admin
Step 3

Auction Settings

Tune settings in the Settings tab before starting. These can be adjusted mid-auction too.

⏱

Timer

How many seconds each player gets. Default: 30s

⬆️

Bid Increment

How much each standard bid raises the price. Default: 500 pts

🔔

Bid Bump

Seconds added to timer when a new bid comes in. Default: +10s

✅

Bid Confirmation

When enabled, teams see a confirm dialog before each bid registers — prevents accidental taps.

⏱ Timer Mode

When the countdown hits zero, the player is automatically sold to the highest bidder (or marked unsold if no bids).

Best for fast-paced auctions.

🔨 Manual Mode

Timer is advisory only. When it hits zero, admin clicks Hammer to close the sale. Admin can also hammer early.

Best for controlled auctions.

Budget constraint

The Bid button auto-disables if bidding would leave you unable to fill your remaining squad slots at minimum bid. You can never overbid.

👑 Admin
Managing Edge Cases

Handling Special Situations

✕ No bids received

If no team bids before the timer expires, the player is automatically marked Unsold (Timer mode) or admin uses Mark Unsold (Manual mode).

↩ Re-auction with price override

In Auction Controls → Unsold Players, click Re-Auction. An inline editor appears — adjust the base price if needed, then Confirm to re-queue.

⏸ Pause for discussion

Use the Pause button to freeze the timer mid-bid — useful if there's a dispute or technical issue. Resume continues from where it left off.

↩ Rollback last sale

In Auction Controls, click Rollback Last Sale to undo the most recent transaction — player returns to the pool and the team's budget is refunded.

⚠️ Don't redeploy or restart the server mid-auction without downloading a State JSON backup first (Settings tab → ⬇ Export State).

⚠ Reset Auction

Clears all bids and resets team budgets, but keeps your league config and player list. Use this to re-run the same auction.

🗑 Full Reset

Wipes everything — players, teams, config — back to factory defaults. Requires admin password. Use with caution.
