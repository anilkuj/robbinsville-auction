⭐ Highlights
⚡

Server-Authoritative Real-Time

All bids and timers are controlled server-side via Socket.io. Every participant sees the exact same state instantly — no polling, no drift.

🔨

Two Auction End Modes

Switch between Auto Timer (player sells when countdown hits zero) and Manual Hammer (admin decides exactly when to close each sale).

👑 Admin Features
⚙️

League Configuration

Set number of teams, squad size, starting budget, and minimum bid. All configurable before the auction begins.

🏊

Custom Player Pools

Define pools (A1, A2, B1 … D) with individual base prices and player counts. Add or remove pools freely.

👥

Team Management

Create teams with custom names and passwords. Credentials are shared with team owners to log in on their devices.

📥

CSV Player Import

Download a pre-filled CSV template, add player names, and upload. Server validates counts per pool automatically.

▶️

Auction Controls

Next Player, Pause, Resume, Mark Unsold — full control over the auction flow from a single admin panel.

↩️

Re-Auction Players

Any unsold player can be sent back to the auction pool and re-bid on at any point during the event.

⏱️

Configurable Timer

Set the countdown duration per player. A bid bump automatically adds seconds when a new bid comes in.

📊

Results Export

Download a CSV of all players — sold price, team, pool, and status — for post-auction records and announcements.

💾

State Backup & Restore

Export full auction state as JSON at any time. Auto-restored from disk on server restart — never lose progress.

🏏 Team Features
📱

Any Device

Works on phones, tablets, and laptops. No app install needed — just a browser.

🔴

Live Auction View

See the current player, countdown timer, highest bid, and bid history update in real time.

💰

Smart Bid Button

One-tap bidding. Auto-disabled if you can't afford it or your squad is already full.

📋

Squad & Budget Tracker

Live sidebar shows your roster, remaining budget, and how many slots are left to fill.

🔒 Rules & Guardrails
🚫

Budget Enforcement

Server blocks any bid that would leave a team unable to fill remaining squad slots at minimum bid price.

🙅

No Self-Outbidding

Teams cannot bid when they are already the highest bidder. Prevents accidental double-bids.

🔐

JWT Authentication

Every session is secured with a signed token. Team credentials are isolated — teams only see their own data.

👤 Login Roles
👑

Admin

Full access — league setup, auction controls, player import, exports, reset

🏏

Team Owner

Bidding view — live auction, bid button, squad tracker, bid history

👁️

Spectator-ready

Any screen can show the live auction view — great for a projector display
